YouTube Downloader Pro
A powerful, modern desktop application for downloading YouTube videos, playlists, and audio with ease.

Features
Single & Playlist Downloads - Download individual videos or entire playlists/channels
Built-in YouTube Search - Search and download directly from the app
Audio Extraction - Convert to MP3, FLAC, WAV, M4A, and more
Quality Options - From 144p to 4K with format selection (MP4, MKV, WebM)
Fast Downloads - aria2c integration with 16 concurrent connections
Batch Processing - Queue multiple URLs for sequential downloading
SponsorBlock - Auto-skip sponsor segments
Cookie Support - Access age-restricted content via browser cookies
Dark/Light Theme - Modern CustomTkinter interface
Download History - Track and re-download previous videos
Installation
Prerequisites
Python 3.8+
FFmpeg
